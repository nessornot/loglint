# LogLint

LogLint - плагин для `golangci-lint`, который анализирует логи и следит за их форматом и безопасностью. Поддерживаются только `log/slog` и `go.uber.org/zap`.

## Правила линтера

1. Логи должны начинаться со строчной буквы
2. Логи должны быть только на английском языке
3. Логи не должны содержать спецсимволы или эмодзи
4. Логи не должны содержать потенциально чувствительные данные

## Установка и запуск

```
# Сборка бинарника custom-gcl (займет около 10 секунд)
golangci-lint custom

# Запуск по конкретной директории
./custom-gcl run ./quickTest/
```
